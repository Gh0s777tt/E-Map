## 2. Bezpieczeństwo — 🟢 A−

Brak **P0/P1**. Zakres po v0.47 (orders/invoices/invoice_items/documents/driver_user_link) czysty:
SELECT przez `is_member_of`, zapis przez `has_role(owner/dispatcher)`, RPC z `SECURITY DEFINER` +
`SET search_path` + jawnym `has_role`. Storage `documents`: bucket prywatny, polityki wiążą
pierwszy segment ścieżki z `company_id` aktywnego membera (brak cross‑tenant).

- **P2 — `order_set_status` dopuszcza status `assigned` dla kierowcy** ([0032](../supabase/migrations/0032_order_assignment.sql)): kierowca może „cofnąć" postęp. Rekomendacja: ograniczyć do `('in_progress','delivered')`. Ryzyko niskie (własne zlecenie, audytowane).
- **P2 — brak rate‑limitu na odczyt PIN karty** ([0025](../supabase/migrations/0025_fuel_card_pin_guard.sql)): odczyt audytowany i zgodny z modelem (kierowca płaci w automacie), ale brak licznika nadużyć.
- **P3** — brak `middleware.ts` (auth egzekwowany per‑route + RLS, spójne); triggery audytu używają `auth.uid()` (NULL przy ewentualnej zmianie service‑role — dziś nie występuje).

✓ Sekrety: `.gitignore` blokuje `.env*`/`*.key`/`*.pem`; brak zahardkodowanych kluczy. Service‑role tylko serwerowo (`createSupabaseAdminClient`, import w `app/api/**` + `lib/push.ts` z `server-only`). Brak PII/PIN w logach.

## 3. Wydajność — 🟠 C+

- **P1 — `/stats` ładuje całe tabele klient‑side** ([stats/page.tsx](../apps/web/app/(app)/stats/page.tsx)): `listFuelLogs` 2000 ×2 + trips 2000 + `listOrders` (bez limitu); rentowność/anomalie/alerty/trend liczone w przeglądarce, choć wykresy biorą tylko 6–8 ostatnich okresów. Rekomendacja: filtr `from`/`to` (już wspierany w `listFuelLogs`) + agregaty per‑miesiąc w widoku/RPC Postgres.
- **P1 — `listOrders`/`listInvoices` bez `.limit()`** ([orders.ts](../packages/api/src/data/orders.ts), [invoices.ts](../packages/api/src/data/invoices.ts)): cała historia firmy do pamięci klienta. Rekomendacja: paginacja / domyślny limit / okno czasu.
- **P1 — `/monthly` pobiera 5000 ×2 wierszy paliwa** ([monthly/page.tsx](../apps/web/app/(app)/monthly/page.tsx)) na 1–6 miesięcy. Rekomendacja: `from = monthsEndingAt(month, 6)` zamiast `limit: 5000`.
- **P2 — brak indeksu `invoices(order_id)`** ([0030](../supabase/migrations/0030_invoices.sql)): RPC łączą faktury po `order_id` (jest tylko `invoices_company_idx`). Rekomendacja: `create index … on invoices(order_id)`.
- **P3** — N+1 nie znaleziono (wszędzie `Promise.all`); indeksy list głównych obecne i trafne; maplibre lazy‑loaded; `<img>` tylko dla QR (uzasadnione).
